# Semiproject

자바만을 사용하여 만든 [스파이더맨]컨셉의 미사일 슈팅게임 입니다.

방향키와 컨트롤키, 스페이스바키를 통해 조작하며, 적군의 공격을 피하고 적군을 물리쳐 점수를 쌓아 목표 스테이지에 도달하면 승리하는 게임 방식입니다.
